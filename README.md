# Diwali_sales_analysis
Exploratory Data Analysis (EDA) of Diwali sales data using Python, Pandas, Seaborn, and Matplotlib to uncover customer purchasing behavior, demographic trends, and sales insights.
This project focuses on analyzing Diwali sales data to understand customer behavior and purchasing patterns. Using Python-based data analysis and visualization techniques, the project highlights key insights such as gender-wise spending, age group distribution, and overall sales trends during the Diwali festival.
